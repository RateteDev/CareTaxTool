# CareTaxTool

医療費控除申請のための領収書解析ツール

![demo](./README/demo.mp4)

## ツールページ

https://caretaxtool.ratete.dev

## 概要

医療費の領収書をアップロードすると、自動で内容を解析し、医療費控除申請に必要な情報を抽出します。

## 使用方法

1. API Keyを設定
2. 領収書画像をアップロード
3. 解析結果を確認・エクスポート
